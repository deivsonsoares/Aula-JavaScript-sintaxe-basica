ESTRUTURAS CONDICIONAIS

São intruções para realizar determinadas tarefas a partir de uma condição, seja de decisão ou repetição;

Ex: Um jogo precisa mudar o placar toda vez em que um jogador marca pontuação;

LAÇOS DE REPETIÇÃO

São estruturas condicionais que repetem uma instrucao ate atingir determinada condição

- FOR;
Funciona como uma repetição de instrução ate que a condição seja falsa
- FOR/IN;
Executa repetição a partir de uma propriedade
